# 面试刷题平台

一个基于 Next.js + Spring Boot 的面试刷题平台，集成了 FastGPT 智能问答功能。

## 核心功能

### 基础功能
- 用户注册登录
- 题库管理
- 在线刷题
- 题目搜索

### AI 智能问答 (FastGPT)
- 支持新建 AI 对话
- 支持多轮对话历史记录
- 支持按分类查看历史对话
- 支持流式和非流式响应

## 技术架构

### 前端
- Next.js + React 18
- Redux 状态管理
- Ant Design 组件库

### 后端
- Spring Boot
- MySQL + Redis
- FastGPT API 集成
